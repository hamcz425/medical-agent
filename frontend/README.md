# 医疗RAG系统数据管理前端

基于React + TypeScript + Redux的医疗大语言模型RAG系统数据管理前端界面。

## 功能特性

- 📊 **数据概览** - 系统统计信息、健康状态监控
- 📁 **文档管理** - 医疗文档的增删改查、分类管理、搜索功能
- 🤖 **RAG查询** - 基于检索增强生成的智能问答系统
- 📈 **状态管理** - Redux状态管理，数据流清晰

## 技术栈

- **前端框架**: React 18
- **类型系统**: TypeScript
- **状态管理**: Redux + Redux Thunk
- **UI组件库**: Ant Design 5
- **构建工具**: Webpack 5
- **路由**: React Router 6

## 项目结构

```
src/
├── api/              # API接口封装
├── components/       # 公共组件
├── pages/            # 页面组件
│   ├── Dashboard.tsx    # 数据概览页
│   ├── Documents.tsx    # 文档管理页
│   └── RAGQuery.tsx     # RAG查询页
├── store/            # Redux状态管理
├── types/            # TypeScript类型定义
├── App.tsx           # 根组件
└── index.tsx         # 入口文件
```

## 核心功能

### 1. 数据概览仪表盘
- 系统统计信息展示（总文档数、已处理文档、查询次数等）
- 系统健康状态监控
- 最近上传文档列表

### 2. 文档管理
- 支持多种文档类型（病历、研究论文、药物信息、临床指南）
- 文档CRUD操作
- 文档状态跟踪（待处理、已处理、已索引）
- 全文搜索功能

### 3. RAG智能查询
- 基于医疗知识库的智能问答
- 查询置信度展示
- 参考来源追溯
- 查询历史记录

## 开发亮点
